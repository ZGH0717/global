### Button

### 使用指南
```html
import { Button } from 'global-vue';

Vue.use(Button);

```
### 基本用法
```html
<global-button size="btn-lg" type="btn-primary-color">
  确认
</global-button>
```
### Attributes

| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| size  | 按钮大小    | String   | — | btn-sm |
| type  | 按钮类型    | String   | — | btn-default |
