### Cell

### 使用指南
```html
import { Cell } from 'hua-ui';

Vue.use(Cell);

```
### 基本用法
```html
<hua-cell>
  内容
</hua-cell>
```
### 显示右边小箭头
```html
<hua-cell is-arrow>
  内容
</hua-cell>
```
### Attributes

| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| is-arrow  | 是否显示右边小箭头    | Boolean   | — |  false |
