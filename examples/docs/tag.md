### Tag

### 使用指南
```html
import { Tag } from 'iv';

Vue.use(Tag);

```
### 基本用法
```html
<i-tag type="tag-default">确认</i-tag>
```
### Attributes

| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| type  | 按钮类型    | String   | — | tag-default |
